### 说明

- 平时开发需要经常切换JDK版本，可用CMD命令进行快速切换。

- 该脚本中涉及修改JDK、MAVEN环境变量。
- 我的MAVEN配置文件中配置了使用的JDK编译版本，所以该脚本中同时切换MAVEN。
